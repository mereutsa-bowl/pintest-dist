# pintest-dist — раздача PinTest (тест)

Только дистрибутив, **без исходного кода** (код — в приватном `pintest-project`).

- `license.json` — манифест: allowlist Device-ID, версии, ссылка на APK.
- APK — в Releases.

Управление — утилитой `tools/pintest-license.sh` из основного репозитория
(добавить/отозвать/продлить устройство, выкатить версию).

APK привязан к устройству: без записи в `license.json` приложение не запустится.
